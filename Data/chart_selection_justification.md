# Chart Selection Justification

## Sales Trend View
Question Answered:
How are sales changing over time?

Chart Type:
Line Chart

Reason:
Line charts clearly display trends across time periods.

## Regional Performance View
Question Answered:
Which regions generate the highest sales and profit?

Chart Type:
Bar Chart

Reason:
Bar charts allow easy comparison between regions.

## Category Profitability View
Question Answered:
Which categories are most profitable?

Chart Type:
Bar Chart

Reason:
Bar charts highlight differences in profitability across categories.

## Customer Segment View
Question Answered:
How do customer segments compare?

Chart Type:
Bar Chart

Reason:
Provides clear comparison between segments.

## Shipping Performance View
Question Answered:
Which shipping modes have longer delivery times?

Chart Type:
Bar Chart

Reason:
Simple comparison of average delivery days.

## Discount vs Profit View
Question Answered:
How does discount affect profitability?

Chart Type:
Scatter Plot

Reason:
Scatter plots show relationships between two numerical variables.

## Return Analysis View
Question Answered:
Which categories have higher returns?

Chart Type:
Bar Chart

Reason:
Allows easy comparison of return volume across categories.

## Design Principles Applied
- Clear titles
- Consistent formatting
- Minimal clutter
- Interactive filters
- Business-focused layout
